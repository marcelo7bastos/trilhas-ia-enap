# Trilhas de IA na EV.G/Enap

Guia independente de curadoria para orientar a escolha de programas e formações em inteligência artificial, dados e tecnologia disponíveis na Escola Virtual Gov (EV.G/Enap).

O objetivo é ajudar pessoas servidoras e equipes públicas a escolherem uma trilha conforme o próprio contexto: começar rápido, aplicar IA na rotina administrativa, liderar iniciativas ou aprofundar-se em TIC, dados e tecnologia.

## Página

- Página principal: [`index.html`](index.html)
- Versão legada: [`legacy/preview.html`](legacy/preview.html)

## Aviso

Este projeto não é uma página oficial da Enap ou da EV.G. Ele é uma curadoria independente baseada em páginas públicas da Escola Virtual Gov.

Para inscrição, público-alvo, certificação, carga horária, disponibilidade e ementa, prevalecem sempre as páginas oficiais da EV.G.

Informações verificadas em 28/05/2026.

## Publicação no GitHub Pages

Depois de enviar este repositório para o GitHub:

1. Abra o repositório no GitHub.
2. Entre em `Settings > Pages`.
3. Em `Build and deployment`, selecione `Deploy from a branch`.
4. Em `Branch`, selecione `main` e a pasta `/root`.
5. Salve.
6. Aguarde a publicação e abra a URL exibida pelo GitHub Pages.

## Manutenção

Ao atualizar o guia:

1. Revise as páginas oficiais da EV.G.
2. Atualize cargas horárias, títulos e links quando necessário.
3. Atualize a data de verificação no `index.html` e neste `README.md`.
4. Faça um novo commit descrevendo a alteração.
